_Loads a ushort as an argument._

### Syntax ###
```cs
load_ushort(value)
```

| Argument | Type | Description |
| --- | --- | --- |
| value | double | The ushort to load. |

**Returns:** 1

### Description

This function loads a ushort as an argument.
