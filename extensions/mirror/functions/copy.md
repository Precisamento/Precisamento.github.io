_Copies an instance reference, and returns the new id._

### Syntax ###
```cs
copy(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance to be copied. |

**Returns:** Instance Id

### Description

This function copies an instance reference, and returns the new id.
