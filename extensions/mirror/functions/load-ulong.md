_Loads a ulong as an argument._

### Syntax ###
```cs
load_ulong(value)
```

| Argument | Type | Description |
| --- | --- | --- |
| value | double | The ulong to load. |

**Returns:** 1

### Description

This function loads a ulong as an argument.
