_Returns the bool saved by the specified id._

### Syntax ###
```cs
get_saved_bool(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance. |

**Returns:** Bool

### Description

This function returns the bool saved by the specified id.
