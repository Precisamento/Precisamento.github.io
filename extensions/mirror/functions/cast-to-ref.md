_Converts an instances type to its ByRef form._

### Syntax ###
```cs
cast_to_ref(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance to have it's type converted. |

**Returns:** 1

### Description

This function converts an instances type to its ByRef form.
