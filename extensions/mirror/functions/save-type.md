_Saves the specified type and returns its id._

### Syntax ###
```cs
save_type(type_name)
```

| Argument | Type | Description |
| --- | --- | --- |
| type_name | string | The name of the type to save. |

**Returns:** Instance Id

### Description

This function saves the specified type and returns its id.
