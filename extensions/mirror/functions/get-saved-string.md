_Returns the string saved by the specified id._

### Syntax ###
```cs
get_saved_string(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance. |

**Returns:** String

### Description

This function returns the string saved by the specified id.
