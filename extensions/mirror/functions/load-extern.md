_Loads a saved instance as an argument._

### Syntax ###
```cs
load_extern(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance. |

**Returns:** 1

### Description

This function loads a saved instance as an argument.
