_Saves the type of the specified instance._

### Syntax ###
```cs
save_instance_type(id)
```

| Argument | Type | Description |
| --- | --- | --- |
| id | double | The id of the instance to get the type from. |

**Returns:** Instance Id

### Description

This function saves the type of the specified instance.
