_Loads an sbyte as an argument._

### Syntax ###
```cs
load_sbyte(value)
```

| Argument | Type | Description |
| --- | --- | --- |
| value | double | The sbyte to load. |

**Returns:** 1

### Description

This function loads an sbyte as an argument.
