_Gets the string at the given index in the split._

### Syntax ###
```cs
_split_get_index(split_id, index)
```

| Argument | Type | Description |
| --- | --- | --- |
| split_id | real | The id of the split to use. |
| index | real | The index of the string to get. |

**Returns:** String
