_Gets the size of the split in bytes. Used to create a buffer of the appropriate size._

### Syntax ###
```cs
_split_get_size(split_id)
```

| Argument | Type | Description |
| --- | --- | --- |
| split_id | real | The id of the split to use. |

**Returns:** Real
