_Transfers data from a split to a buffer._

### Syntax ###
```cs
_split_fill_buffer(split_id, buffer)
```

| Argument | Type | Description |
| --- | --- | --- |
| split_id | real | The id of the split to use. |
| buffer | ptr | The memory address of the buffer. |

**Returns:** 1
